1. 初识Vue

  • 创建Vue实例 , 且要传入一个配置对象

  • root容器里的代码依然符合html代码规范 , 只不过混入了一些特殊的Vue语法

  • root容器里的代码被称为 Vue模板

  • 注意：

    • Vue实例和容器是一一对应的:
    • 真实开发环境中只有一个Vue实例 , 且配合组件一起使用
    • 中的XXXX要写js表达式 , 且XXXX会自动读取到data中的所有属性
    • 一旦data中的数据发生改变 , 那么模板中用到该数据的地方也会自动更新

  • ```html

    初识Vue 
    <!-- 引入Vue -->
<script src="../js/vue.js" type="text/javascript"></script>

    <!-- 盒子容器-->
<!-- 插值语法 -->
<div id="root">
    <h1>hello ,{{name}},来自{{address}} !!!</h1>
</div>

<script type="text/javascript">
    Vue.config.productionTip = false;

    // 创建Vue实例
    new Vue({
        // el用于指定当前Vue实例为那个容器服务 , 值通常为css选择器
        el: "#root",
        // data用于存储数据 , 数据供el所指定的容器去使用
        data: {
            name: "张三",
            address: "云南"

        }

    })
</script>

3. 数据绑定

  • 绑定数据的两种形式：

    • 单向绑定（v-bind）： 数据只能从data流向页面

    • 双向数据绑定(v-model:value): 数据不仅能从data流向页面，还可以从页面流向data

  • 注意：

    • 双向数据绑定一般是用在表单类元素上
    • v-model:value <==等价==> v-model=”” ， 因为v-model默认收集value的值
  • <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>数据绑定</title>
    <script src="../js/vue.js"></script>
</head>

<body>

    <div id="root">
        <!-- 双向绑定只能应用于表单类元素 (简化前)-->
        <!-- 
        单向数据绑定: <input type="text" v-bind:value="name"><br> 
        双向数据绑定: <input type="text" v-model:value="name"> 
        -->
        <!-- 双向绑定只能应用于表单类元素 (简化后)-->
        单向数据绑定: <input type="text" :value="name"><br> 
        
        双向数据绑定: <input type="text" v-model="name">

    </div>


    <script type="text/javascript">
        new Vue({
